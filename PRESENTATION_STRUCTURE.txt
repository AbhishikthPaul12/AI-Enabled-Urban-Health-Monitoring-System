SLIDE 1: INTRODUCTION (3 minutes)
================================================================================

Project Title: MedAxis Health Manager
Subtitle: AI-Enabled Urban Health Monitoring System

Problem Statement:
- Healthcare management is fragmented across multiple platforms
- Patients lack centralized health tracking systems
- Doctors need efficient patient management tools
- Administrators require system oversight capabilities
- No integrated platform connecting patients, doctors, and admins

Proposed Solution:
- Web-based integrated health management platform
- Role-based access control (Admin/Doctor/Patient)
- Real-time health monitoring and analytics
- Centralized appointment and prescription management

Project Objectives:
1. Develop a comprehensive health management system with multi-role support
2. Integrate Oracle database with web interface using JDBC
3. Implement secure authentication using SHA-256 password hashing
4. Demonstrate complete CRUD operations across all modules
5. Deploy using Spring Boot with embedded Tomcat server
6. Create responsive UI with dark mode support

SLIDE 7: CHALLENGES AND SOLUTIONS (4 minutes)
================================================================================

CHALLENGE 1: Database Connection Management
Problem: Frequent database connection failures and resource leaks
Solution: Implemented connection pooling using Spring DataSource
          Configured proper resource cleanup in finally blocks
Result: Stable database connections with automatic management

CHALLENGE 2: Session Security
Problem: Unauthorized access to protected pages via direct URL
Solution: Created SessionFilter to validate user sessions
          Implemented role-based access control checks
Result: Secure navigation with proper authentication enforcement

CHALLENGE 3: Role-Based Navigation
Problem: Sidebar showing wrong links for different user roles
Solution: Dynamic sidebar rendering based on session role attribute
          Created separate JSP pages for each role
Result: Role-appropriate navigation across all pages

CHALLENGE 4: Dark Mode Persistence
Problem: Dark mode preference lost on page refresh
Solution: Implemented localStorage to save user preference
          Added JavaScript to apply theme on page load
Result: Persistent dark mode across all pages

CHALLENGE 5: Chart.js Integration
Problem: Charts not displaying dynamic data from database
Solution: Embedded JSP variables into JavaScript
          Used AJAX for real-time data updates
Result: Interactive charts showing actual health trends

CHALLENGE 6: Password Hashing Mismatch
Problem: Login failing due to different hash formats
Solution: Standardized SHA-256 with Base64 encoding
          Updated all existing passwords in database
Result: Consistent authentication across all accounts

CHALLENGE 7: JSP Import Errors
Problem: Classes not found when importing in JSP
Solution: Embedded JDBC code directly in JSP pages
          Used fully qualified class names
Result: Successful database operations from JSP
